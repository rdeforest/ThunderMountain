- [ ] **Hour 1**: Natural break point identified

### Recovery Tools
**Stun-lock phrases**:
- "Give me one second to process that brilliance"
- "That's so good I need a moment"
- "Let me check my notes on that"

**Energy management**:
- Delegate combat to players ("Kyle, help me run this")
- Take stretch breaks
- Use prepared materials to reduce cognitive load

**Consistency catches**:
- Turn into worldbuilding opportunities
- "Why WOULD the NPC have said that?"
- Note for later clarification

## Session Planning Workflow

### 1. Review Current State (Day -7)
- Read campaign-state.md
- Check timers and active hooks
- Review player tendencies from past sessions

### 2. Brainstorm Session Arc (Day -5)
- Identify 2-3 potential story threads
- Plan main encounter/challenge
- Prepare contingencies for player choices
- Consult with Claude for worldgen if needed

### 3. Create Session Directory (Day -3)
```
notes/session-N/
├── 00-INDEX.md          # Control panel
├── opening-scene.md     # How to start
├── hooks/               # Available plot threads
│   ├── hook-1.md
│   └── hook-2.md
└── quick-refs/          # Stat blocks, DDB links
    └── stat-blocks.md
```

### 4. Prepare Materials (Day -1)
- Write lecture opening
- Create NPC cards
- Link stat blocks
- Test technical setup
- Run mental walkthrough

### 5. Final Review (Session Day)
- Reread 00-INDEX.md
- Review NPC voices
- Check physical materials
- Set up space early
- Relax and trust preparation

## Emergency Protocols

### "They Did Something Unexpected"
1. Call for break to think
2. Ask clarifying questions
3. Say "Let me check how that would work"
4. Consult notes or improvise within established lore
5. Trust "Yes, and..." philosophy

### "I'm Losing Energy"
1. Call for 10-minute break
2. Delegate next combat to players
3. Use pre-written material to coast
4. Consider early wrap if needed (30min short okay)

### "Rules Dispute"
1. Make quick ruling for momentum
2. Note to look up later
3. Establish house rule if needed
4. Consistency over correctness

### "Players Are Stuck"
1. Environmental pressure escalation
2. NPC arrives with information
3. Doom clock advancement consequence
4. "What does [PC name] think about this?"
